# Earn a Pro License

If Serial Studio is useful to you but purchasing a Pro license is not an option, you can earn one by making meaningful contributions to the project. This keeps the tool improving while giving you full access in return.

## Ways to Contribute

### Make It Known

- Write a post or tutorial about Serial Studio on a developer forum (Reddit, CSDN, Hackaday, etc.)
- Create a demo video on YouTube, Bilibili, or similar platforms
- Share on social media or community channels and provide proof of reach (engagement metrics, screenshots)

### Help Extend Hardware Support

- Send binary logs or protocol data for unsupported devices
- Donate development hardware (boards, analyzers, adapters)

### Improve the Tool

- Translate the UI or documentation into other languages
- Write or improve documentation, help pages, or code comments
- Submit pull requests for bug fixes, new features, or platform support

## What You Get

Based on your contribution, you will receive a Pro license key:

| Contribution Type              | License Duration |
|--------------------------------|------------------|
| Forum post / tutorial / video  | 6 months         |
| Hardware donation              | 12 months        |
| Protocol log / sample data     | 3-6 months       |
| Translation / documentation    | 6-12 months      |
| Code contribution (merged PR)  | 6-12 months      |

Contributions must be original, useful, and verifiable.

## How to Submit

1. Open a GitHub Discussion with the `value-for-license` tag.
   - Title: `Request: Pro License via [Your Contribution Type]`
   - Include a description of what you did (links, files, screenshots).
   - Provide the email address where the license key should be sent.

2. Wait for review. The maintainer will respond within a few days.

3. If approved, your license key will be issued directly.

## Getting Started with Code Contributions

1. Fork the repository on GitHub.
2. Read the project guidelines and code style documentation in the repository.
3. Pick an open issue or propose a new feature.
4. Submit your pull request.
5. Once merged, request your Pro license via GitHub Discussions.

## Why This Program Exists

Serial Studio is built and maintained by a single developer. Contributions of any kind -- visibility, hardware, translations, or code -- directly support the project's long-term sustainability. If you cannot pay, contribute. Help spread the word.
