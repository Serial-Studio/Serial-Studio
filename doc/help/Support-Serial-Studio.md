# Support Serial Studio

Serial Studio started as a passion project and is now used by thousands of developers, engineers, educators, and makers around the world. It's still maintained by a single developer, with no corporate funding or venture capital behind it.

Any form of support directly funds development, testing on real hardware, documentation, and faster releases.

## Ways to help

### Buy a Pro license

The most direct way to support the project. A Pro license unlocks MQTT, Modbus, CAN Bus, MDF4 export, 3D visualization, and other advanced features, and it funds ongoing development. If you use Serial Studio at work, your purchasing or IT department can handle the transaction.

Store: <https://store.serial-studio.com/>

### Become an affiliate

If you create content for embedded developers, makers, educators, or hardware teams, you can support the project by referring customers and earning a commission on successful sales. It's a practical option for bloggers, YouTubers, newsletter authors, and community managers who already recommend tools to their audience.

Affiliate applications are handled through Lemon Squeezy: <https://store.serial-studio.com/affiliates>

### Star the repository

Help others find the project by starring it on GitHub. It costs nothing and bumps visibility in search results and trending lists.

Repo: <https://github.com/Serial-Studio/Serial-Studio>

### Sponsor development

Support development through GitHub Sponsors or PayPal donations. Links are on the project's GitHub page.

PayPal: <https://www.paypal.com/donate?hosted_button_id=XN68J47QJKYDE>

### Report issues

Improve quality by reporting bugs and suggesting improvements on GitHub Issues. Include steps to reproduce, expected behavior, and platform details.

### Contribute code

Pull requests for bug fixes, features, translations, or documentation are all welcome. Meaningful contributions can also qualify for a free Pro license (see [Earn a Pro License](Earn-Pro-License.md)).

### Share your projects

Show off what you've built with Serial Studio. Real-world use cases inspire other people and help the community grow.

### Write about it

Blog posts, tutorials, YouTube videos, and conference talks help reach new users who could use the tool.

## Why it matters

Serial Studio is real software, built during unpaid hours because it needs to exist. No marketing tricks, no tracking, no gimmicks. Whether you're a student or a company, your support helps keep practical, privacy-respecting tools alive.
